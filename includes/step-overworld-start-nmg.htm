Wake Up

Get bomb and fire bomb at start

  1. During the opening cutscene, obtain the free bomb and free fire bomb. Be sure to wait two seconds between codes.
  2. Run north, past the warp pad and the mailbox and up the two sets of stairs.
Optimization: You can get a 2-3 second head-start by using the Plushie code to interrupt the opening cutscene. However, the code is 29 inputs, and if you type it slower, you save less time.
