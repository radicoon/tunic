Plush Start

Use the "Plushie Code" to start early and get bomb

  1. As soon as the in-game timer starts, type in the Plushie code.
  2. While the fox shows the plushie after waiting two seconds between codes, obtain the free firecracker.
  3. Leave while the camera is still zooming.
This is a very annoying code and saves 2-3 seconds. It's up to you!
